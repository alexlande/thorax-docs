Getting Started

Thorax 2 is presently in beta, the stable source and documentation are still available.

Thorax can be used standalone in any JavaScript environment in addition to the boilerplate projects provided above.

var view = new Thorax.View({
  template: "Hello world!"
});
view.render();
$("body").append(view.el);
